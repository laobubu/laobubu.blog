好吧，于是我有必要在假期结束前搞一个啥东西表示这个假期做过什么东西。正巧遇到新机器的 DNS 被 GFW 污染得体无完肤，Flickr 各种无法显示图片，Hosts 文件又没有及时更新，所以写了个输入域名生成 Hosts 文件的小php程序。而且可以通过 POST 请求来生成 Hosts 文件（学校里的OpenWRT路由器有救了，wget 支持 POST）。

程序和源代码

立即进入： http://heroku.laobubu.net/hostsgenerate/

在线源码(GitHub)： https://github.com/laobubu/HostsGenerator

Features

  • Load domains from Hosts file, HTML source codes and your plain text files.
  • Filter to avoid some lines like 127.0.0.1 localhost
  • Specify the address of some domains (to block ads?)
  • Choose your DNS server
  • TCP query
  • IPv6 (AAAA) query
  • Generate Hosts by posting data. (Really friendly to some cheap OpenWRT devices)
