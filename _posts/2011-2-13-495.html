清华大学真的是一个很牛的地方，他们在IPv4枯竭之时和mozilla合作了，推出了一个叫做“紫狐”的浏览器。

此浏览器实际上也就是一个火狐浏览器而已，不过它不同的就是：拥有自己的Hosts列表。也就是说大多数IPv6网站都被使用IPv6访问。

紫狐浏览器下载地址还有很多，比如新浪的就有一枚。

IPv6Tool.bat

这个不是紫狐的东西，是我编写的。功能嘛……提升IPv6在你电脑上的优先级、为你的电脑安装IPv6功能。

BAT文件，纯文字的，不带病毒，不信自己检查：



ECHO →启动IPv6Helper服务 net start 6to4 ECHO →安装IPv6 ipv6 install ECHO →隧道连接 netsh int ipv6 set teredo client teredo.ipv6.microsoft.com ECHO →提高IPV6优先级 netsh int ipv6 set prefix ::1/128 50 0 netsh int ipv6 set prefix ::/0 40 1 netsh int ipv6 set prefix 2002::/16 30 1 netsh int ipv6 set prefix ::/96 20 3 netsh int ipv6 set prefix ::ffff:0:0/96 10 4 netsh int ipv6 set prefix 2001::/32 5 1 ECHO →正在等待连接（这里的ping是为延时所用 :) ） ping 1.0.0.0 -n 1 -w 10000 ECHO →隧道信息 netsh int ipv6 show teredo ECHO →IP信息（请自己检查） ipconfig /all




保存为一个bat文件（实际上是文本文件的一类吧）即可
